# Stamp Windows Installation Guide

Welcome to **Stamp**! Follow these easy steps to install the professional standalone application on your Windows PC.

## Prerequisites
-   **Operating System**: Windows 10 or 11 (64-bit recommended).
-   **Disk Space**: Approximately 200MB.

## Step-by-Step Installation

1.  **Download**: Obtain the `Stamp Setup 1.0.0.exe` installer from your distribution source.
2.  **Launch**: Double-click the `.exe` file to start the setup wizard.
3.  **Permissions**: If prompted by Windows User Account Control (UAC), click **Yes** to allow the installer to make changes.
4.  **License Agreement**: (Optional, if added) Read and accept the terms.
5.  **Installation Location**: 
    -   The default path is `C:\Users\YourName\AppData\Local\Programs\Stamp`. 
    -   You can click **Browse...** to choose a different folder if you prefer a custom location.
6.  **Shortcuts**:
    -   The installer will automatically create a **Desktop Shortcut** for easy access.
    -   A shortcut will also be added to your **Start Menu**.
7.  **Complete**: Once the progress bar finishes, you can check the box "Run Stamp" and click **Finish**.

## Tips for a Better Experience
-   **Pin to Taskbar**: Right-click the app icon (once it's running) and select **Pin to taskbar** for one-click access.
-   **Updates**: If you download a new version, simply run the new installer; it will update your existing installation automatically.

> [!NOTE]
> This application is hardware-bound. Ensure you have your activation key ready if this is your first time launching the app on this device.
